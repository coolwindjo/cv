### Glossary of Technical Terms

||||
|:-----------|:---------------------------------------------------------|
| **Model Quantization** | A technique to reduce the size and computational load of deep learning models. Quantization script converts the model to lower-bit integer formats (e.g., 8-bit, 4-bit). The system will automatically compare the quantized model's accuracy, inference speed, and memory usage against the original model. If the accuracy drop exceeds a predefined threshold, the build fails. |
| **ABA**, **Active Brake Assist** | 능동 제동 보조 시스템. 전방 충돌 위험 시 충돌 회피 및 사고 저감을 위해 차량 감속 명령을 보내는 시스템입니다,. ABA는 **AEB (Automatic Emergency Braking)**와 함께 언급되며, 충돌 위험 시 경고 및 제동을 발생시키는 기능을 포함합니다,,. MPC 5.5의 **ASIL B 인증**을 받은 기능 중 하나입니다,,. |
| **AEB**, **Advanced Emergency Brake** / **Automatic Emergency Braking** | 선행 차량과의 충돌 위험을 예측하여 비상 제동을 적용해 사고를 방지하거나 완화하는 시스템입니다,. ABA의 핵심 구성 요소입니다. |
| **APTR**, **Adaptive PowerTrain Request** | 운전자가 설정한 속도와 거리를 유지하며 주행 모드 및 전방 상황에 따라 가속 및 제동을 보조하는 시스템입니다,. 이는 **ACC (Adaptive Cruise Control)** 기능과 관련이 깊습니다. |
| **ACC**, **Adaptive Cruise Control** | 운전자가 설정한 속도나 거리를 도로 상황에 따라 유지하고, 내비게이션 경로를 따라 주행하도록 보조하는 시스템입니다,. APTR의 주요 기능에 해당합니다. |
| **ESS**, **Emergency Stop System** | 운전자가 운전 불능 상태라고 판단될 경우, 차량을 차로에 유지하며 안전하게 비상 정차시키는 시스템입니다,. AEB와 달리 점진적으로 감속을 수행합니다. |
| **LKA**, **Lane Keeping Assist** | 차선 이탈이 예상될 때 조향 개입을 통해 차량을 차선 안쪽으로 복귀시켜 사고를 방지하는 시스템입니다,,. MPC 5.5의 **ASIL B 인증**을 받은 기능 중 하나입니다,,. |
| **LKAS**,**Lane Keeping Assist System** | 차선 유지 보조 시스템을 통칭하며, 조승현 님이 Botts' dots 감지 컴포넌트를 개발한 기능이 이에 해당됩니다. |
| **LDW**, **Lane Departure Warning** | 차선 이탈이 예상되거나 이탈할 때 운전자에게 경고(시각/촉각/청각)를 제공하는 시스템입니다,,. |
| **HUD**, **Head-Up-Display** | 운전자에게 차량 정보를 전면 유리에 투영하여 제공하는 시스템입니다. MPC 5.5에서는 HUD의 밝기 조절이나 **AR HUD**를 위한 캘리브레이션 정보를 제공했습니다,. |
| **TSA**, **Traffic Sign Assist** / **Traffic Sign Recognition (TSR)** | 교통 표지판을 인식하여 운전자에게 정보를 제공하고, 자동으로 설정 속도를 변경하여 사고를 방지하는 시스템입니다,,. |
| **IHC**, **Intelligent Head-Light Control** | 주변 차량을 감지하여 눈부심을 주지 않도록 상향등을 부분적으로 하향등으로 조정하는 기능입니다,. **HBA (High Beam Assist)**와 유사합니다,. |
| **EME**, **Ego-Motion Estimation** | 차량의 **자체 모션 추정** 알고리즘으로, 인지 기능 및 제어 기능에 필요한 정확한 자차 위치 및 움직임 정보를 실시간으로 제공합니다,. |
| **ASIL**, **Automotive Safety Integrity Level** | ISO 26262 기능 안전 표준에 따른 안전 무결성 등급으로, MPC 5.5는 **ASIL B** 인증을 획득했습니다,,,,. |
| **ASPICE**, **Automotive Software Process Improvement and Capability dEtermination** | 자동차 소프트웨어 개발 프로세스 개선 및 역량 평가 모델이며, MPC 5.5 프로젝트는 **Capability Level 3 (CL3)** 인증을 받았습니다,,. |
| **ADTF**, **Automotive Data and Time-Triggered Framework** | 자동차 데이터 및 시간 트리거 프레임워크로, 조승현 님은 이를 활용하여 기능 신호 시각화 도구 구현 및 차선 감지 정확도 평가 시스템을 개발하셨습니다,,. |
| **IDPS**, **Intrusion Detection and Prevention System** | 침입 탐지 및 방지 시스템. 현재 조승현 님께서 담당하고 계신 자동차 사이버 보안 분야의 핵심 모듈입니다,. |
| **RTCU**, **Roadside/Remote Telematics Control Unit** | 차량과 외부 시스템 간의 통신을 담당하는 장치입니다,. |
| **SoP**, **Start of Production** | 제품의 양산 시작 시점을 의미합니다,,. |
| **D-AUTO**, **Deceleration Auto** | 회생제동을 적극 사용하도록 운전자에게 악셀 페달을 떼라고 유도하는 점을 보면, "자동 감속(Automatic deceleration)"의 성격도 포함되어 있어, 사내 명명 규칙에 따라 "Drive Auto" 또는 "Deceleration Auto" 같은 표현을 D-AUTO 표기 |
